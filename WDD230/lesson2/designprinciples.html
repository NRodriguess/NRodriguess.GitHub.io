Visual Principles

Amazon

amazon.com

On this site they apply the principle of Visual Hierarchy on most pages, which allows the user to realize the things that matter most on the page. In the case of this page they used a prominent button written "Try prime" more prominent by using good color and size to attract the user's attention. The biggest eye-catcher is the blue banner and then the big button, making it possible for the user to be attracted to where the site wants.

White space and clean design

Amazon

amazon.com

White space or negative space is a space on the website where the user can rest their eyes. Most sites contain a lot of information but it is not always good to fill the page with things, causing users to be lost. Even on this Amazon shopping site, users were able to have this blank space and be calmer when deciding where to browse.
